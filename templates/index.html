👋 Welcome to Project Samarth!

I use RAG (Retrieval-Augmented Generation) with sentence transformers and FAISS vector search to understand complex natural language queries—completely offline!

✨ Capabilities:
  • Multi-source synthesis: Compare rainfall + list top crops in one query
  • District-level analysis: Find highest/lowest producing districts
  • Trend correlation: Analyze production patterns with climate data
  • Policy insights: Generate data-backed recommendations
  • Fuzzy matching: Handles typos (try "Maharastra" → "Maharashtra")

🚀 All processing powered by local embeddings—no external APIs!
